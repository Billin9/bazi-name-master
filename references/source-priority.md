# Source Selection Guide

Do not treat source choice as a rigid ranking exercise.

The model should choose sources dynamically based on:

- the chart's structural needs
- the naming temperament the user wants
- whether the source line naturally supports the final name

The goal is not to force every name back to the same few classics. The goal is to choose the source family whose language, imagery, and moral-aesthetic tone best match the child and the naming direction.

Use these as matching hints, not as hard rules:

- Need `清正、温雅、端稳`:
  - `《诗经》`
  - `《论语》`
  - `《礼记》`

## Citation Rules

For every candidate name, provide:

- source text name
- the specific original line or phrase
- a short semantic reinterpretation that explains why the source supports the name

Do not cite a classic only by title. The connection must be traceable.

## Source Selection Guidance

- Prefer sources whose imagery aligns with the chart, the user's naming goal, and the chosen temperament.
- Prefer lines whose semantic force is clear and naturally transferable into a name.
- If two sources both work, choose the one that gives the cleaner meaning, stronger atmosphere, or better naming dignity.
- `《尔雅》` and `《本草纲目》` are often better as semantic support or imagery support than as the main literary出处.
- `《韩非子》` can be used, but usually only when the desired temperament genuinely benefits from clarity, severity, or hard rational structure.
- Buddhist sources are valid when the intended atmosphere is genuinely clear, empty, serene, or luminous; do not use them by default just to sound elevated.
- Avoid forcing a citation onto a name that does not naturally fit the source.

## Tone Guardrails

- Do not use grandiose quotations only for prestige.
- Do not choose a source whose original context creates negative or ambiguous connotations.
- When using Zhou Yi, pre-Qin texts, or Buddhist texts, explain the naming meaning in plain language.
